# MyrCore

这是一个基于 rCore Tutorial 实现的简易操作系统

## 功能

- [x] 外部中断支持，包含时钟中断、键盘中断
- [x] 物理内存和虚拟内存支持：内存分配和页面置换
- [x] 多进程和多线程支持
- [x] 设备树和rCore的 `Simple File System` 文件系统支持，标准输入输出
- [x] 简单的系统调用和用户程序支持

working in the future:

- [ ] shell 支持
- [ ] 更多的系统调用
